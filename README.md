# README Bricks

**누구나 간편하고 쉽게 README.md 파일을 만들 수 있습니다!**

## 사용 방법

- 먼저 왼쪽 Templates 탭에서 각종 템플릿을 고릅니다.
- 그러면 중앙 코드 에디터에 자동으로 붙여넣기가 됩니다.
- 그리고 설명에 따라 적절하게 수정합니다.
- 오른쪽 Results 탭에서 실시간으로 결과물을 확인합니다.
- 끝입니다! 간편하죠?

## 세부 사용 방법

- 당연히 더 있습니다.
- 먼저 각 템플릿 오른쪽에 <img src="src/assets/modify.svg" width="20"/>
  버튼이 있습니다. 이 버튼을 클릭할 경우 각종 템플릿을 수정할 수 있습니다!

#### # 자신만의 커스텀 템플릿은 못 만드나요?

- 아닙니다! <img src="src/assets/multiple.svg" width="20"/>
  버튼을 클릭하시면 커스텀 템플릿을 만들 수 있습니다.
- 그 후 자유롭게 수정 및 삭제가 가능합니다!

#### # 으악 잘못 만들었는데 초기화는 불가능한가요?

- 가능합니다. <img src="src/assets/refresh.svg" width="20"/>
  버튼을 클릭하시면 아주 쉽게 초기화 할 수 있습니다.

#### # 전부 만들었는데 복사는 어떻게 하나요?

- 위에 있는 copy 버튼 클릭 한 번 이면 복사가 됩니다.
- 이제 붙여넣기만 하면 완료!
